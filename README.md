# 🎯 Secret Auction Program

A simple and beginner-friendly **Secret Auction Program** built using Python.  
This project allows multiple users to place secret bids, and the program determines the highest bidder at the end.

---

## 🚀 Features

- 🧑 Multiple bidder support
- 💰 Stores bid amounts using Python dictionaries
- 🏆 Automatically finds the highest bidder
- ✨ Clean and professional console output
- 🎯 Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Python 3
